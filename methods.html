Pipeline Overview

We integrate TF annotations, promoter sequences, and motif scanning to construct gene regulatory networks. The core inference methods include:

  • PANDA for global network inference
  • LIONESS for sample-specific networks
  • Motif Scanning using FIMO on 1kb upstream promoter regions

Genomic data is primarily sourced from Ensembl Fungi. We preprocess sequences, annotate TFs, and score motif-target associations.

Reproducibility

All scripts and workflows are available on our GitHub repository. Full versioning and parameter documentation ensures transparency.
