REST API

*Need to implement API*

Use our RESTful API to query gene and network data.

GET /api/gene/:id

curl https://spore-db.org/api/gene/creA

GET /api/network/:species

curl https://spore-db.org/api/network/a_nidulans

GET /api/search?q=term

curl https://spore-db.org/api/search?q=carbon

Authentication

No authentication is currently required. Rate limiting applies.
